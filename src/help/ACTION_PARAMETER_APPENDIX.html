Action Parameter Appendix

Scope

This appendix completes the action coverage started in:

  • ACTION_PARAMETER_MATRIX.md

It focuses on the remaining actions found in:

  • Enums.cs
  • CommonUtility.cs
  • CommonPage.cs
  • helper files under BromCom.Tests\Utilities\PageObjectModelHelper

Reading Rules

  • Element and ElementCategory still follow the rules in the main matrix.
  • This appendix only describes the remaining action contracts.
  • If a row says copy-existing-only, the action is too page-state-specific to author safely from theory alone.
  • If a row says enum-only, the action exists in Enums.cs but no runtime branch was found in the scanned execution path.
